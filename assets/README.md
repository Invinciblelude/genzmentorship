# Assets

Place your app assets here:

- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon
- `favicon.png` - Web favicon

For now, the app will work without these assets, but you should add them before publishing.

You can generate these using tools like:
- [Expo Asset Generator](https://www.npmjs.com/package/expo-asset-generator)
- [App Icon Generator](https://www.appicon.co/)
